# Engineering Portfolio

This portfolio showcases three independent projects demonstrating skills in **simulation, computation, and hands-on engineering**.  

- **CFD Flow Optimisation:** Analysed and optimised flow around an airfoil using Python-based CFD tools.
- **Neural Network from Scratch:** Implemented a fully-connected neural network to predict engineering outcomes from simulation data.
- **3D Print Prototype:** Designed, iterated, and tested a small physical prototype using CAD and 3D printing.

Each project includes code, results, and documentation to demonstrate **problem-solving, technical ability, and engineering design thinking**.
