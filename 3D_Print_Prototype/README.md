## Objective
Design and test a small physical prototype that demonstrates engineering concepts.

## Tools
- CAD software (Fusion 360, SolidWorks, or FreeCAD)
- 3D Printer
- Python / MATLAB for design calculations (optional)

## Method
1. Designed initial prototype in CAD.
2. Printed first iteration; tested functionality (e.g., flow test, mechanical movement).
3. Iterated design based on performance and printing issues.
4. Documented final design with photos and diagrams.

## Results
- Photos of printed prototype
- Notes on design changes and reasoning
- Optional video demonstration of prototype in action

## Lessons Learned
- Importance of print tolerances and material choice.
- Iterative design process improves functionality.
- Combining simulation results with physical testing validates engineering assumptions.
